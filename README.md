## How to Run the Project

### Requirements

- Docker
- Docker Compose
- `make`

### Start the project

From the repository root, run:

```bash
make up
```

Open in browser
Frontend: <http://localhost:5173>
Backend health check: <http://localhost:3000/api/health>
